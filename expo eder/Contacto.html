Nivel de datos

El nivel de datos, a veces denominado nivel de base de datos, nivel de acceso a datos o backend, es donde se almacena y administra la información procesada por la aplicación. Puede ser un sistema de administración de bases de datos relacionales, como PostgreSQL, MySQL, MariaDB, Oracle, DB2, Informix o Microsoft SQL Server.
